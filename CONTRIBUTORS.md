## How this file is used

- It records meaningful contributions that the maintainers want to make explicit.
- It helps preserve human-readable credit even when GitHub's default graph does not attribute a change as expected.
- It does not replace the GitHub PR history, commit history, or contributors graph.

## Attribution notes

- GitHub's contributors graph only counts merged work on the default branch.
- GitHub also requires the commit author email to be linked to the contributor's GitHub account for attribution in the graph.
- Generic or shared bot emails can prevent correct attribution.
- Commit co-authors can also be credited when GitHub can map the co-author email to an account.

For contribution workflow and attribution guidance, see [CONTRIBUTING.md](CONTRIBUTING.md).
